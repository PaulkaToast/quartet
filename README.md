# quartet
An online multi-person sequencer that you can play around with to make music

## Build Instructions

1. Go into the client directory 

2. `npm install` 

3. ` ./node_modules/.bin/webpack`

## Building the Database

`./create-tables.sh`

## Starting the Server

`./run-server.sh`

## Emptying the Database

`./drop-tables.sh`
